# Book Tracker - Трекер прочитанных книг

**Автор:** Иван Иванов

**Вариант:** Book Tracker

**Дата сдачи:** 15.05.2026

---

## Описание программы

Book Tracker — это графическое приложение для ведения списка прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц, фильтровать их по жанру и количеству страниц, а также удалять записи. Все данные автоматически сохраняются в JSON-файл.

Программа решает проблему учета прочитанной литературы, помогает систематизировать книги и быстро находить нужные по фильтрам.

---

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, os) - устанавливать ничего не нужно, они уже есть в Python

Дополнительные библиотеки не требуются.

---

## Как запустить

```bash
git clone https://github.com/твой-логин/book-tracker.git
cd book-tracker
python book_tracker.py

Как пользоваться
Добавление книги:

Заполните все поля (Название, Автор, Жанр, Количество страниц)

Нажмите кнопку "Добавить книгу"

Книга появится в списке

Удаление книги:

Выберите книгу в списке (кликните по ней)

Нажмите кнопку "Удалить книгу"

Фильтрация книг:

По жанру: введите текст в поле "Фильтр по жанру" - список обновится автоматически

По страницам: выберите в выпадающем списке ">200" (книги где больше 200 страниц) или "<200" (книги где меньше 200 страниц)

Сброс фильтров:

Нажмите кнопку "Сбросить фильтры" - все фильтры очистятся

Просмотр всех книг:

Все добавленные книги отображаются в списке

Счетчик внизу показывает общее количество книг
